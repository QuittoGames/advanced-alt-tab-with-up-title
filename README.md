# GNOME Advanced Alt-Tab Customization

This project contains my personal GNOME configuration. I customized my desktop environment and wanted to use the **Advanced Alt-Tab Window Switcher** extension from:

https://github.com/G-dH/advanced-alttab-window-switcher

My goal was to create an Alt-Tab experience closer to Windows 11, especially keeping the application title and logo visible together, something that GNOME does not provide by default.

During the customization process, I realized that achieving this layout was not straightforward. The default behavior made it impossible to keep the window title at the top together with the application icon/logo in the way I wanted, which made the interface feel disconnected from the Windows-style workflow I was trying to reproduce.

Instead of manually analyzing the entire extension code, I decided to use an AI coding agent to investigate possible solutions. In this case, I used **OpenCode with DeepSeek V4 Flash Free** to analyze the problem and find where the modification could be implemented.

Surprisingly, the agent was able to identify the necessary changes and produce a working solution. This was my first real experience with "vibe coding" — using AI as a development partner to explore, modify, and improve a project without directly writing every line myself.

After seeing the result working, I decided to publish this configuration on GitHub to make it easier to transfer between machines and allow other people who want a similar GNOME experience to use it.

Huge thanks to **G-dH** for creating the original **Advanced Alt-Tab Window Switcher** extension. This project would not exist without their work and the excellent foundation provided by the extension.
